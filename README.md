# data-analysis-1
Machine learning and data analysis projects in Python using Pandas, Scikit-learn, and Matplotlib, including classification, exploratory data analysis, and customer segmentation.
## Overview

This repository contains several data science projects covering the complete machine learning workflow:

- Data cleaning and preprocessing
- Exploratory Data Analysis
- Feature engineering
- Data visualization
- Classification models
- Clustering analysis
- Model evaluation

The projects were completed using Python and common data science libraries to solve practical analytical problems.

---

# Projects

## 1. Student Performance Analysis

A machine learning project that investigates factors affecting students' academic performance.

### Key Tasks

- Data preprocessing
- Missing value handling
- Correlation analysis
- Exploratory Data Analysis
- Feature selection
- Classification using Support Vector Machine (SVM)
- Random Forest classification
- Confusion matrix evaluation
- Model performance comparison

### Technologies

- Pandas
- NumPy
- Matplotlib
- Scikit-learn
